=======   MixMods.com.br   ======= - Best visualization with monospaced fonts, like "Consolas".


------- Instructions:

Extract the "PortableGTA" folder to the ModLoader folder.
Compatible with SA, VC and III.

ATTENTION: If you use Improved Fastloader, install Improved Fastloader in the root folder of the game (outside of ModLoader),
          otherwise, the function to automatically start on the last loaded save game will not work.

-- Download the latest version of Modloader: MixMods.com.br/2015/01/SA-Modloader.html


 Now, your saved game, settings etc. will be in the "userfiles" folder in your game folder.
 Never have the game installed in the "Program Files" folder.
 The game will open without your saved games and settings for controls, graphics etc,
 You may want to copy the contents of your current User Files folder (in Documents) to the "userfiles" of this mod to preserve settings.
