------- Instructions:

Extract the folder "New Horse Mod (cavalo)" to your Modloader folder.

-- Download the latest version of Modloader: MixMods.com.br/2015/01/SA-Modloader.html
-- For color variations, use PedFuncs: MixMods.com.br/2018/11/PedFuncs.html



 See "Local.jpg" for the spawn locations.
 You can use the common commands, like sprint and jump.
 To neigh, use the jump button without running.
